Array Madness

Given two integer arrays a and b, both of length >= 1, create a program that returns true if the sum of the squares of each element in a is strictly greater than the sum of the cubes of each element in b.

Example:

      arrayMadness([4, 5, 6], [1, 2, 3]);

      // returns true since 4 ** 2 + 5 ** 2 + 6 ** 2 > 1 ** 3 + 2 ** 3 + 3 ** 3
